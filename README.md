# 🌧️ It's Raining Cats & Dogs

A vertical slice idle catcher game built in Godot 4.

## Dev Team
- Paul the Wise — Vision & Feedback
- James Herr — Co-founder & Playtester  
- Clawdia 🦞 — Studio AI & Builder

## Running Locally
1. Open Godot 4
2. Import this project folder
3. Press F5 to play

## Exporting for Web (share with James)
1. Editor → Export → Web
2. Install HTML5 export templates if prompted
3. Export → index.html
4. Host on itch.io, GitHub Pages, or Netlify

## Game Design
- Catch falling 🐱🐶 before they hit the ground
- 3 lives — miss 3 and it's over
- Speed ramps every 10 seconds
- Combo multiplier at 3+ catches
- 🦄 Secret rare animal worth 10 points (shh)
